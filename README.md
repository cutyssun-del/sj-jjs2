# 신정개발 급여 대시보드

㈜신정개발 직원 급여 정산용 웹 대시보드입니다. **엑셀 파일(`public/data/payroll_sample.xlsx`)을 데이터 원본**으로 읽어 급여대장·급여명세서·4대보험·세금을 브라우저에서 보여줍니다.

> ⚠️ 저장소에 포함된 데이터는 **공개 배포용 가상(샘플) 데이터**입니다. 실제 직원 개인정보가 아닙니다.

## 기능

- **요약** : 재직 인원, 연간 지급/실지급/공제 총액, 부서별 집계
- **급여대장(연간)** : 직원·월 필터. 직원을 선택하면 그 직원의 1~12월 급여가 한눈에
- **급여명세서** : 직원·월 선택 → 발송/인쇄용 명세서 (PDF 저장 가능)
- **4대보험·세금** : 국민연금·건강·장기요양·고용보험, 소득세·지방소득세 공제 내역

## 기술 스택

Next.js 14 (App Router) · React 18 · TypeScript · SheetJS(xlsx) — 별도 DB 없이 엑셀만으로 동작.

## 로컬 실행

```bash
npm install
npm run dev      # http://localhost:3000
```

## 배포 (Vercel)

1. 이 저장소를 [vercel.com](https://vercel.com) 에서 **Import**
2. Framework: **Next.js** 자동 인식 → 그대로 **Deploy**
3. 배포 완료 후 발급되는 URL 접속

## 데이터 갱신 방법

`public/data/payroll_sample.xlsx` 파일을 교체하고 커밋/푸시하면 Vercel이 자동 재배포되어 대시보드가 갱신됩니다. 엑셀의 **급여대장** 시트 열 순서를 유지해야 합니다(파싱 기준: `lib/payroll.ts`).
