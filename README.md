# 신정개발 경영지원 대시보드

㈜신정개발 직원 급여 정산용 웹 대시보드입니다. **급여관리 마스터 엑셀 파일을 데이터 원본**으로 읽어 급여대장·급여명세서·4대보험·세금을 브라우저에서 보여줍니다.

## 🔒 보안 설계 (가장 중요)

이 서비스는 **엑셀 파일을 저장소나 서버에 올리지 않습니다.**

- 사용자가 **자기 컴퓨터에서 엑셀을 선택** → **브라우저 메모리 안에서만** 파싱됩니다.
- 파일은 서버·GitHub·인터넷 어디로도 **업로드·전송되지 않습니다.** (순수 클라이언트 처리)
- 새로고침하면 데이터는 사라집니다. 어디에도 저장되지 않습니다.
- 모든 스프레드시트 형식(`*.xlsx, *.xlsm, *.xls, *.csv` 등)과 `public/data/`는 [`.gitignore`](.gitignore)로 차단되어 **실수로라도 커밋되지 않습니다.**
- 공개 데모는 엑셀이 아니라 코드에 내장된 **가상 데이터**([`lib/demoData.ts`](lib/demoData.ts))를 씁니다. 실제 개인정보가 아닙니다.

## 기능

- **요약** : 재직 인원, 연간 지급/실지급/공제 총액, 부서별 집계
- **급여대장(연간)** : 직원·월 필터. 직원을 선택하면 그 직원의 1~12월 급여가 한눈에
- **급여명세서** : 직원·월 선택 → 발송/인쇄용 명세서 (PDF 저장 가능)
- **4대보험·세금** : 국민연금·건강·장기요양·고용보험, 소득세·지방소득세 공제 내역

## 기술 스택

Next.js 14 (App Router) · React 18 · TypeScript · SheetJS(xlsx) — 별도 DB 없이 사용자 엑셀만으로 동작.

## 로컬 실행

```bash
npm install
npm run dev      # http://localhost:3000
```

## 배포 (Vercel)

1. 이 저장소를 [vercel.com](https://vercel.com) 에서 **Import**
2. Framework: **Next.js** 자동 인식 → 그대로 **Deploy**
3. 배포 완료 후 발급되는 URL 접속 → &lsquo;내 엑셀 파일 선택&rsquo;으로 사용

## 사용 방법

1. 배포된 사이트 접속
2. **&lsquo;내 엑셀 파일 선택&rsquo;** 클릭 → 급여관리 마스터 엑셀(`.xlsx`) 선택
3. 대시보드가 즉시 표시됨. 엑셀에서 급여를 수정한 뒤 다시 선택하면 갱신됩니다.

> 엑셀의 **급여대장** 시트 열 순서를 유지해야 합니다(파싱 기준: [`lib/payroll.ts`](lib/payroll.ts)).

## 향후 보안 강화 옵션

- 접근 제한(로그인/비밀번호) 적용
- 조직 내부망 전용 배포 또는 Vercel 접근 보호(Password Protection)
